Tipos de Posicionamiento en CSS

Posición Relativa

El elemento se posiciona relativamente a su posición normal. Se mueve respecto a donde debería estar originalmente.

Relativo

Posición Absoluta

El elemento se posiciona de forma absoluta respecto a su contenedor más cercano con posición relativa.

Absoluto

Posición Fixed

El elemento se mantiene fijo en la pantalla incluso al hacer scroll. Siempre visible en la misma posición.

Fixed

Posición Sticky

El elemento se comporta como relativo hasta alcanzar un punto de desplazamiento, luego se "pega" como si fuera fijo.

Sticky
